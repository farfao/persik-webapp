# 📸 Персик Premium - Web App для сканирования состава

## 🚀 Деплой на Vercel

### 1. Установить Vercel CLI
```bash
npm i -g vercel
```

### 2. Зайти в папку webapp
```bash
cd webapp
```

### 3. Деплой
```bash
vercel --prod
```

### 4. Получить URL
После деплоя вы получите URL вида:
```
https://your-app.vercel.app
```

## 🔧 Настройка в боте

В файле `persik_premium.py` замените:
```python
web_app=WebAppInfo(url="https://your-domain.com/webapp/", title="📸 Сканирование состава")
```

На ваш URL:
```python
web_app=WebAppInfo(url="https://your-app.vercel.app/", title="📸 Сканирование состава")
```

## 📱 Функции Web App

- ✅ **Доступ к камере** - задняя камера телефона
- ✅ **OCR распознавание** - Tesseract.js (русский + английский)
- ✅ **Фильтрация INCI** - извлечение только состава
- ✅ **Отправка в бота** - через Telegram WebApp API
- ✅ **Адаптивный дизайн** - работает на всех устройствах

## 🎯 Как работает

1. **Пользователь нажимает** "📸 Сканировать состав"
2. **Открывается Web App** с камерой
3. **Наводит на состав** и нажимает "Сканировать"
4. **OCR распознает текст** с изображения
5. **Фильтрует состав** - оставляет только INCI
6. **Отправляет в бота** - для анализа
7. **Бот анализирует** - и показывает результат

## 🔍 Технологии

- **Frontend**: HTML5 + CSS3 + JavaScript
- **OCR**: Tesseract.js 4.0
- **API**: Telegram WebApp API
- **Деплой**: Vercel (бесплатно)
- **Стили**: Glassmorphism дизайн

## 💡 Особенности

- 🎨 **Современный дизайн** - градиенты, blur, анимации
- 📱 **Мобильная оптимизация** - идеальна для телефонов
- 🔒 **Безопасность** - данные только внутри Telegram
- ⚡ **Быстрое сканирование** - мгновенная обработка
- 🌍 **Мультиязычность** - русский и английский

## 🚀 Запуск

После деплоя на Vercel:
1. Обновите URL в боте
2. Перезапустите бота
3. Нажмите "📸 Сканировать состав"
4. Тестируйте сканирование!
